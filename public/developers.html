How do I register as a developer?

In order to submit applications to the store, you need to register as a developer. Registration is free and can be done here.


What resources are available to me?

The store allows installed applications to access its TOR connectivity so they too can increase the privacy of network connections they make. Documentation for this is provided here.

Additionally, developers can combine this TOR connectivity with a HTTP library developed by the store team to connect to backend services securely. Documentation for this is provided here.
