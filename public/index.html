What is the sTORe?

The sTORe is a mobile application store where you can browse and install mobile applications designed with privacy in mind.

At the moment, the sTORe is only available on Android devices running Android 4.1 (Jellybean) or higher.


Installing the store

You can download the latest version of the store here.

Please download or transfer the application to your phone to install it.
