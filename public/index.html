What is the sTORe?

The sTORe is a mobile application store where you can browse and install mobile applications designed with privacy in mind.

You can install the sTORe on any Android device running Android 4.1 (Jellybean) or higher.



Download the sTORe

Download the sTORe for your Android device here.

Remember to enable "Unknown sources" from within the "Security" section of your "Settings" app to allow the sTORe and other apps to be installed.
